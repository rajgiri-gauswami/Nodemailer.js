## Installation

Install my-project with npm

```bash
  cd Nodemailer.js
  
  npm install

  npm run start
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`MAIL_USER`

`PASS_KEY`
